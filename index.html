Git flow


Utilizando Git


git status

git diff

git log

En resumen: comandos esenciales para comprobar el estado

  • git status muestra los archivos con cambios respecto al repositorio
  • git diff muestra las diferencias de los cambios respecto al repositorio
  • git log muestra el histórico de commits de la rama actual

git add

git commit

Resumen: comandos para incluir archivos en el repo

  • git add mueve archivos al staging area
  • git commit envía al repositorio los cambios en el staging area

git branch

Permite mostrar/crear una nueva rama desde la actual

git checkout

Permite mover la cabeza (HEAD) y cambiar de rama/commit

Resumen: comandos referentes a ramas

  • git branch permite mostrar/crear una nueva rama desde la actual
  • git checkout permite mover la cabeza (HEAD) y cambiar de rama/commit
  • git merge permite fusionar una rama dentro de otra
